Python helpers for the OpenRazer Keyboard
==================================================

Examples
---------

Set brightness to 50

    from openrazer import Keyboard
	kb = Keyboard()
	kb.brightness = 50

Set mode to reactive

    from openrazer import Keyboard
	kb = Keyboard()
	kb.mode_reactive('ff00ff')
